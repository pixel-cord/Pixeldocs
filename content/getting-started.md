# Getting Started

## Where to Start?

Welcome to the Pixelcord documentation! This guide will help you get started with installing and using Pixelcord.

## What is Pixelcord?

**Pixelcord** is a fork of [Equicord](https://github.com/Equicord/Equicord) (which is itself a Vencord fork): a Discord client mod that ships with over 300+ plugins. You can find the source, including all of the official plugins, in [our GitHub repository](https://github.com/pixel-cord/pixelcord). Official Pixelcord plugins live under `src/pixelcordplugins/`, while Vencord-based plugins live under `src/plugins/`.

**Pixeltop** is Pixelcord's standalone desktop client (a fork of Equibop) with Pixelcord pre-installed. There's no need to patch your existing Discord installation — just download it and go. Grab the latest build from the [Pixeltop releases page](https://github.com/pixel-cord/Pixeltop/releases).

Pixelcord supports **Linux**, **macOS**, and **Windows**.

## Choosing How to Install

There are two ways to get Pixelcord running:

1. **Pixeltop** — the easiest option. Download a prebuilt desktop client with Pixelcord already included. See the [Installation](/installation) guide.
2. **Build from source** — clone the repository, build it yourself, and inject it into your existing Discord install. This is required if you want to write or run your own plugins. See [Building from Source](/building-from-source).
